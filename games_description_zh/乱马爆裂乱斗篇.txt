根据著名动漫《乱马1/2》改编的一款对战格斗游戏。游戏的操作比较另类，共有两种操作形式可供选择，TYPE B基本上和其他正统的格斗游戏的操作是一样的，TYPE A则要靠按下跳跃键来进行跳跃，第一次玩的人会很不适应，所以推荐大家选用TYPE B这种操作形式来玩这款游戏。 游戏中的跳跃分以下几种: 1.垂直跳跃 TYPE A：角色处于站立状态时，上 +跳跃键 TYPE B： 输入上方向 . 2.后跳 TYPE A：角色处于站立状态时， 后上或者后 +跳跃键 TYPE B： 输入后上方向. 3.前跳 TYPE A：角色处于站立状态时，前上 或者前 +跳跃键 TYPE B： 输入前上方向. 游戏中登场的角色都是原著中的经典人物，游戏画面一般，人物动作生硬，不过好在游戏的操作非常简单，只需简易的指令，角色便可发出各种不同的必杀技攻击。另外游戏中即使防御普通攻击也会消耗体力的。
